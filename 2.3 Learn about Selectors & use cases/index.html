Types of Selectors: 1. html tag selector eg:(h1,p) etc.. 2. class selector: (.) 3. id selector: (#) 4. universal selector: (*) 5. group selector..
1. html tag selector :

welcome guys eg: h1{ color: blue; }
2. class selector: (.)

hello everyone

Note: An class name cannot start with a number! eg: eg: .heading1{ color: red; }
3. id selector:(#)

good morning

Note: An id name cannot start with a number! eg: eg: #subheading{ color: blueviolet; }
4.Universal selector:(*) *{ color:red }
